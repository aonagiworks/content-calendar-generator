# Content Calendar Generator

Generator ide konten sederhana untuk UMKM, content creator, dan bisnis kecil.

## Fitur
- Pilih niche bisnis
- Pilih jumlah hari konten
- Generate ide konten otomatis
- Format konten
- CTA
- Download hasil ke CSV

## Niche yang tersedia
- Coffee Shop
- Skincare
- Fashion
- Makanan Rumahan
- Jasa Desain
- Bengkel
- Toko Aksesoris
